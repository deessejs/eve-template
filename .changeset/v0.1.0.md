Initial scaffold release. Ships the canonical `agent/` directory (eve 0.24.4
runtime, single model on MiniMax M3 via `vercel-minimax-ai-provider`,
web channel with `[betterAuthFn, vercelOidc(), localDev()]` auth chain),
the Next.js 16 web chat UI (shadcn/ui + Vercel AI Elements + Streamdown +
Shiki), better-auth 1.7.0-rc.1 wired to Drizzle/Postgres with login-only
admission, the two-layer auth boundary (edge `proxy.ts` redirect + DB
re-validation in `app/(authenticated)/layout.tsx`), CLI bootstrap
(`scripts/create-admin.sh` and `scripts/create-user.mjs`), and seven Claude
agent skills that orchestrate the full lifecycle: `spec`, `implement`,
`review-pr`, `create-pr`, `ship`, `release`, plus `triage` and
`create-issue`.
